# Highway 82 Fire — Live Interactive Map

Public, interactive map tracking the Highway 82 Fire (Brantley / Wayne / Glynn Counties, GA, April 2026).

**Live site:** https://naomiwolfe.github.io/highway-82-fire-map/

Auto-rebuilds every 4 hours with the latest fire perimeter, evacuation zones, road closures, and wind forecast.

Sources: Watch Duty, Georgia Forestry Commission, Brantley County Sheriff, Glynn County, NWS.
